What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (like HTML or XML) as a tree of objects, allowing developers to manipulate the content, structure, and style of a webpage dynamically.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript for managing events efficiently in the DOM. Instead of adding an event listener to every individual element that might trigger an event (like clicks), you attach a single event listener to a common ancestor (parent) element. This parent element captures the events that bubble up from its children.
